# NC State University - COS IT
## Initial Install
<ol>
 <li>Open "terminal"</li>
 <li>sudo -s
 <li>mkdir /opt</li>
 <li>cd /opt</li>
 <li>git clone https://github.com/ncsu-cos-it/lnx cos<br>
  <li>cd cos/sbin</li>
  <li>./cos-config</li>
  <li>exit</li>
</ol>

## Manual Config SN/TAG
 <ol>
 <li>Open "terminal"</li>
 <li>cd /opt/cos/etc</li>.
 <li>nano SN.txt<br>
  (Type whatever seriall number you have for computer)</li>
 <li>nano TAG.txt<br>
   (Type whatever asset tag you have for computer)</li>
 </ol>

## Update
<ol>
  <li>Open "terminal"</li>
  <li>cd /opt/cos</li>
  <li>git pull</li>
  <li>exit</li>
</ol>
